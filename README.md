# Space as Time Through Neuron Position Learning
This is the source code for https://arxiv.org/abs/2511.01632. 

## Installation
Follow the instructions in https://genn-team.github.io/genn/documentation/5/installation.html to install PyGeNN.
Clone the following branch: https://github.com/mbalazs98/ml_genn/tree/position_learning
Install mlGeNN with setuptools using pip install . command in the ml_genn directory
